# Output: Format Of The ``xyz`` Crystal Files Created by the ACSD Program

The format for this ``xyz`` file is given as follows:

## First Line

The number of atoms in the full unit cell of the crystal.


## Second Line

Various informaton can be given here, including:

* ``Lattice``: This is the lattice parameters, which are the vectors that describe the unit cell for this crystal.
* ``Properties``: This is the format of the information given in the lines following the second line. In this example, the format is given as the element of the atom, the x, y, and z cartesian positions of the atom, the charge of the atom (given as ``initial_charges``), ``MoleculeList`` is the index of the molecule that atom is apart of in the crystal, and ``NeighboursList`` are the atoms that are bonded to this atom in the molecule (in the crystal).
* ``CrystalSymmetryOperations``: These are the crystal spacegroup symmetry operations for this crystal. 
* ``SameMoleculesDueToCrystalSymmetry``: This identifies which molecules are identical to other molecules in the crystal based on the crystal spacegroup symmetry operations, given by ``SymOpts``. This list is given by --> unique molecule: [list of equivalent molecules based on crystal symmetry]. If there are no equivalent molecules, no list will be given, but the unique molecule name will be given in ``SameMoleculesDueToCrystalSymmetry`` as --> unique molecule:,. 
* ``SolventsList``: This indicates the molecules in the crystal that have been identified as a solvent. 
* ``BondProperties``: This contains all the properties about the bonds between atoms in the crystal. See XXX for more details.
* ``pbc``: Periodic Boundary Conditions in the unit cell vector directions. This should be set to ``"T T T"``, as our crystals should be continuous in all directions. 


### ``BondProperties`` Features

The following information may be given for ``BondProperties``:

* ``bond_type_from_sybyl_type``: This is the Sybyl type for the bond. [Click here](https://tccc.iesl.forth.gr/education/local/quantum/molecular_modeling/guide_documents/SYBYL_data_document.html) under the **Bonds** title for more information about this variable.
* ``involved_in_no_of_rings``: This is the number of rings that this bond is involved in (can be 0, 1, or 2).
* ``is_cyclic``: Is the bond involved in a cyclic bonding scheme, like a bond in a benzene, cyclohexane, or heme in a ring system.
* ``bond_type``: This is the type of bond this is, either a single, double, triple, or aromatic bond. 
* ``is_conjugated``: Is the bond involved in a conjugated system. This will be true if the bond type is double, triple, or aromatic.


## Third Line Onwards

All the atoms given in these lines are for every atom in the crystal unit cell, including the positions of all the symmetrically equivalent molecules. ``...`` is the rest of the atoms in this crystal.

Each line from the third line onwards describes each atom in the crystal. This is given by the ``Properties`` component of the second line. This will include:

* ``species``: This is the element of the atom.
* ``pos``: This is the x, y, and z coordinates of the atom.
* ``initial_charges``: This is the formal charge of the atom. While this is given as a decimal, it will be an integer value.

Other variables that may be included are:

* ``involved_in_no_of_rings``: This is the number of rings that this bond is involved in.
* ``hybridisation``: This is the hybridisation of the atom. This can be either ``sp``, ``sp2``, ``sp3``, or ``-``.
* ``is_H_donor``: This indicates if the atom is a hydrogen-bond donor.
* ``is_H_acceptor``: This indicates if the atom is a hydrogen-bond acceptor.
* ``is_spiro_atom``: This indicates if the atom is a spiro atom. [Click here](https://en.wikipedia.org/wiki/Spiro_compound) for more information about sprio atoms.
* ``MoleculeList``: This indicates the name/number/index of the molecule this atoms is apart of.
* ``NeighboursList``: This lists the indices of all the atoms that are bonded to this atom.
